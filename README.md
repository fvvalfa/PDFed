## ✨ Возможности

### Основные функции

| Функция | Описание |
|---------|----------|
| 📄 **Просмотр PDF** | Открытие и навигация по страницам PDF |
| ✏️ **Добавление текста** | Горизонтальный и вертикальный текст (поворот 90°) |
| 📐 **Настройка шрифта** | Размер от 6 до 72 пунктов |
| 🖱️ **Перемещение** | Свободное перемещение текста по странице |
| ✏️ **Редактирование** | Двойной клик для изменения текста и параметров |
| 🗑️ **Удаление** | Удаление выделенных текстовых аннотаций |
| 🔍 **Масштабирование** | От 10% до 300% |
| 🖨️ **Печать** | С предварительным просмотром |
| 🔤 **TTF шрифты** | Поддержка пользовательских шрифтов |
| 📊 **Информация** | Детальная информация о странице |

### Поддерживаемые типы текста

- **Горизонтальный** — стандартный текст слева направо
- **Вертикальный** — текст сверху вниз с поворотом на 90°

## 📦 Требования

### Системные требования

- **Операционная система**: Windows 10/11
- **Python**: 3.8 или выше
- **ОЗУ**: минимум 512 МБ
- **Диск**: минимум 100 МБ свободного места
